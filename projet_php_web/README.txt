# Application de gestion des tâches en PHP

Cette application Web offre une solution complète pour gérer efficacement une liste de tâches à faire. Elle propose plusieurs fonctionnalités pour vous aider à organiser vos tâches de manière efficace et productive.

## Fonctionnalités principales

1. **Liste des tâches** : Affiche toutes les tâches existantes avec leurs détails, y compris le titre, la description, la date de création, la date d'échéance et le statut.

2. **Ajout de tâches** : Permet d'ajouter de nouvelles tâches à la liste. Vous pouvez spécifier le titre, la description, la date d'échéance et d'autres détails pour chaque tâche.

3. **Mise à jour du statut des tâches** : Vous permet de marquer une tâche comme terminée ou non terminée en cochant une case à côté de chaque tâche dans la liste.

4. **Suppression de tâches** : Vous permet de supprimer une tâche de la liste si elle n'est plus nécessaire ou si elle est terminée.

## Instructions d'utilisation

1. Commencer par executer le script sql (todolist.sql) puis ouvrir le fichier `index.php` dans projet_php  dans votre navigateur. Cela vous dirigera vers la page d'accueil de l'application.

2. Sur la page d'accueil, vous verrez deux choix :
   - "Lister les tâches" : Utilisez ce lien pour afficher la liste des tâches existantes.
   - "Ajouter une tâche" : Utilisez ce lien pour ajouter une nouvelle tâche à la liste.

3. Choisissez l'option souhaitée en cliquant sur le lien correspondant.

4. Suivez les instructions sur chaque page pour effectuer les actions désirées.

5. Pour toute question ou problème, consultez la documentation ou contactez le support technique.

**Note** : Il est recommandé de commencer par consulter le fichier `index.php` pour naviguer dans l'application.

Merci d'utiliser notre application de gestion des tâches !
